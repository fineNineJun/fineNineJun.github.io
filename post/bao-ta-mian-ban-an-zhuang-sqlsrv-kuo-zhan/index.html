宝塔面板安装 sqlsrv 扩展

2022-03-02 2 min read # bt-panel # sqlsrv # php

宝塔面板安装 sqlsrv 扩展，php=7.3

加入源

curl https://packages.microsoft.com/config/rhel/7/prod.repo > /etc/yum.repos.d/mssqlrelease.repo

当前安装的版本为 php7.3 及以上的，如果发现这个链接失效了或者你需要其他版本的，可以到https://packages.microsoft.com/config/rhel/ (opens new window)

安装驱动

yum install msodbcsql mssql-tools unixODBC-devel

下载 pdo_sqlsrv 扩展包

wget http://pecl.php.net/get/pdo_sqlsrv-5.8.1.tgz

更多版本：http://pecl.php.net/package/pdo_sqlsrv (opens new window)

安装 pdo_sqlsrv 扩展

解压扩展包

tar -zxvf pdo_sqlsrv-5.8.1.tgz
cd pdo_sqlsrv-5.8.1

编译安装

/www/server/php/73/bin/phpize
./configure --with-php-config=/www/server/php/73/bin/php-config
make && make install

添加到 php.ini

将扩展添加到 php.ini

echo "extension = pdo_sqlsrv.so" >> /www/server/php/73/etc/php.ini

重启 php-fpm

/etc/init.d/php-fpm-73 reload

检查扩展安装状态或者可以进入phpinfo()中查看

/www/server/php/73/bin/php -m|grep -i  sqlsrv

清理安装包

cd ..
rm -rf pdo_sqlsrv-5.8.1
rm -rf pdo_sqlsrv-5.8.1.tgz
