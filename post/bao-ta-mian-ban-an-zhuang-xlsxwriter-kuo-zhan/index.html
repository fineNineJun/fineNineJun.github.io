宝塔面板安装 xlsxwriter 扩展

· 2022-03-03 Thursday · # php # bt-panel # xlswriter

xlsxwriter 是一个 PHP C 扩展，可用于将文本、数字、公式和超链接写入 Excel 2007 + XLSX 文件中的多个工作表。

点击查看文档

wget https://pecl.php.net/get/xlswriter-1.4.0.tgz

tar -zxvf xlswriter-1.4.0.tgz

cd xlswriter-1.4.0/

/www/server/php/73/bin/phpize

./configure --with-php-config=/www/server/php/73/bin/php-config --enable-reader

make && make install

echo "extension = xlswriter.so" >> /www/server/php/73/etc/php.ini

/etc/init.d/php-fpm-73 reload

/www/server/php/73/bin/php -m|grep -i xlswriter
